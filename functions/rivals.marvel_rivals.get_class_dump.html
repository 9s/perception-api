Function get_class_dump

  • get_class_dump(pointer: number): marvel_rivals.ClassField[]

    Returns a table (array) containing the class fields for the specified class pointer in Marvel Rivals. Each entry in the table includes:

    • name (string) The name of the class field.
    • offset (number) The memory offset associated with the field.
     {
        [1] = { name = "PlayerController::AcknowledgedPawn", offset = 0x120 },
        [2] = { name = "Controller::ControlRotation", offset = 0x154 },
        ...
    }
